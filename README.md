# Run app's services locally

1. Install [Docker](https://www.docker.com), if don't have one
2. Start the app's services by running `docker compose -f docker-compose.dev.yml up -d`

# Installation

```bash
$ npm install
```

# Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
